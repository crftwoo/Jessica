# Technical Context: Stadler Logística Redesign

## Objective
Create a high-end, strategic redesign for Stadler Logística to be presented as a business proposal. The focus is on moving away from an "outdated" look to a "Logistics 4.0" aesthetic that conveys authority, technology, and global reach.

## Tech Stack
- **Framework:** HTML5 / Vanilla Javascript.
- **Styling:** Tailwind CSS (via CDN or local build) for rapid, consistent UI development.
- **Icons:** Lucide Icons (clean, stroke-based).
- **Typography:** Google Fonts (Outfit for headings, Inter for body).
- **Animations:** Framer Motion-inspired CSS transitions and Intersection Observer for scroll effects.

## Core Features
- Responsive Design (Mobile-first).
- Interactive Service Cards.
- Dynamic Statistics Section.
- B2B Lead Capture Form.
- "Screenshot Loop" for visual QA.

## Design Philosophy
Avoid "AI Slop" (generic designs) by using:
1. Custom grid systems.
2. High-contrast typography.
3. Authentic imagery (using real fleet visuals described in research).
4. Micro-interactions that respond to user intent.
